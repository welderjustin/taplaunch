# TapLaunch — Brand Site (v2 modern refresh)

Modernized static site for TapLaunch (NFC keychains + simple sites). Dark, gradient-accented hero, improved buttons/badges, and refined form styles. Deploy anywhere (GitHub Pages, Netlify, Vercel static).

Files
- index.html — Home + contact form
- packages.html — Packages
- faq.html — FAQ
- work.html — Work
- styles.css — Styles (dark theme + accents)

Setup
1) Replace the Formspree action with your form ID in `index.html`
2) Commit this folder to a GitHub repo (e.g., `taplaunch-site`)
3) Deploy:
   - GitHub Pages: enable Pages from main branch
   - Netlify/Vercel: drag‑and‑drop or connect repo (no build step required)

Notes
- Keep logos in `./assets`
- Optional content overrides via `./content/site.json` (existing loader supported)
